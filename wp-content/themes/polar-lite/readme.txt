Polar Lite is a minimal and responsive SuevaFree child theme.

== Description ==

Polar Lite is a minimal and responsive SuevaSuevaFreechild theme, based on Bootstrap framework, with a different layout for elements and sidebar. Now is ecommerce ready and is optimized to work with WooCommerce and Contact Form 7 plugins.

Created by ThemeinProgress, https://www.themeinprogress.com 

Polar Lite WordPress Theme is a child theme of SuevaFree Theme, Copyright 2019 ThemeinProgress.
